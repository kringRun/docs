These pages detail the core interfaces you will use when building with LangGraph. Each section covers a different part of the ecosystem.

## LangGraph

The core APIs for the LangGraph open source library.



- [Graphs](https://reference.langchain.com/javascript/classes/_langchain_langgraph.index.StateGraph.html): Main graph abstraction and usage.
- [Functional API](https://reference.langchain.com/javascript/functions/_langchain_langgraph.index.entrypoint.html): Functional programming interface for graphs.
- [Pregel](https://reference.langchain.com/javascript/classes/_langchain_langgraph.pregel.Pregel.html): Pregel-inspired computation model.
- [Checkpointers](https://reference.langchain.com/javascript/modules/_langchain_langgraph-checkpoint.html): Saving and restoring graph state.
- [MemorySaver](https://reference.langchain.com/javascript/classes/_langchain_langgraph-checkpoint.MemorySaver.html): Storage backends and options.
- [Errors](https://reference.langchain.com/javascript/variables/_langchain_langgraph-checkpoint.ERROR.html): Error types and handling.
- [Channels](https://reference.langchain.com/javascript/classes/_langchain_langgraph.pregel.Channel.html): Message passing and channels.


## Prebuilt components

Higher-level abstractions for common workflows, agents, and other patterns.



- [Agents](https://langchain-ai.github.io/langchainjs/reference/functions/langchain_agents.createAgent.html): Built-in agent patterns.
- [Supervisor](https://langchain-ai.github.io/langgraphjs/reference/modules/langgraph_supervisor.html): Orchestration and delegation.
- [Swarm](https://langchain-ai.github.io/langgraphjs/reference/modules/langgraph_swarm.html): Multi-agent collaboration.
