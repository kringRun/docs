>[DuckDB](https://duckdb.org/) is an in-process SQL OLAP database management system.

## Installation and Setup

First, you need to install `duckdb` python package.

<CodeGroup>
```bash pip
pip install duckdb
```

```bash uv
uv add duckdb
```
</CodeGroup>
