All integrations related to [OpenAI](https://en.wikipedia.org/wiki/OpenAI), an American artificial intelligence (AI) organization.

## Installation and setup

<CodeGroup>
```bash pip
pip install langchain-openai
```

```bash uv
uv add langchain-openai
```
</CodeGroup>

Get an [OpenAI Platform](https://platform.openai.com/docs/overview) API key and set it as the environment variable `OPENAI_API_KEY`.
